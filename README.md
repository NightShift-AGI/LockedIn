# lockedin

**lockedin** is an AI-native, free, open-source replacement for LinkedIn.

The goal is to build a professional network where users can create profiles, share work, discover opportunities, build trust, collaborate, and grow their careers with AI-first workflows.

## Current Stage

Zero stage: idea validation, product planning, architecture planning, team alignment, and MVP definition.

## Suggested MVP Goal

Build a working MVP where users can:

- Create a professional profile
- Post updates
- Follow/connect with others
- Use AI to improve profile and posts
- Discover people, posts, and opportunities
- Run locally as an open-source project
